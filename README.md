# Organizador-Trabalhista
Organizador de documentos desenvolvido para auxiliar no envio de documentação mensal para clientes da empresa. Antes o trabalho manual demorava cerca de 1 a 2 semanas para organizar algo que o programa organiza em menos de 20 minutos e com relatório gerado! Desenvolvido ao longo de 2024, 2025 e início de 2026!
